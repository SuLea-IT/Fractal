这个仓库用于整理“**分形与植物发育/穗部结构**”相关的课题资料、文献、研究方案与阶段性记录。

当前工作重点是：
- 以**水稻、大麦、小麦**为核心对象，比较它们是否适合作为“分形样结构 / 自相似 / 迭代生成”研究体系；
- 将“分形”从单纯形态学描述，推进到**发育规则、分生组织命运转换、分枝层级形成、空间表达与调控网络**层面；
- 围绕潜在目标期刊（如 **Nature Plants / Nature Genetics**）设计更完整的研究方案。

## 当前研究主线

本轮工作的核心认识是：

- “分形”在生物学里不能只理解为视觉上的重复图案；
- 真正重要的是：
  - 是否存在**多尺度重复结构**；
  - 是否存在**迭代式器官发生**；
  - 是否能定位到**分生组织状态转换和调控网络**；
  - 是否能够和**产量、适应性或驯化变异**联系起来。

因此，本仓库中的方案更强调：
**发育机制 + 结构量化 + 空间信息 + 调控变异 + 发表路径设计。**

## 当前推荐判断

截至目前，初步判断如下：

- **水稻**：最适合做“分形样结构 + 发育机制 + 调控变异”的完整故事；
- **大麦**：最适合做“被抑制的重复生成潜力”和分生组织终止机制；
- **小麦**：农艺价值强，但需要避免和已发表的高水平工作直接重复，更适合从 specialized reiteration 角度切入。
